# Bhojnalaya Mauli 🚩
**Authentic Maharashtrian Taste with Vithumauli's Blessings.**

Bhojnalaya Mauli is a premium Pure Veg restaurant web application designed for seamless store pickups. It features a modern UI, WhatsApp-integrated ordering, and a digital invoice system.

## 🌐 Live Demo
Check out the live store here: [https://Rushikesh1881.pythonanywhere.com](https://Rushikesh1881.pythonanywhere.com)

## ✨ Key Features
* **Premium UI/UX:** Clean, responsive design with floating particles and Lottie animations.
* **Smart Ordering:** Quantity selectors and category-wise menu browsing.
* **WhatsApp Integration:** Instant order confirmation and customer feedback sent directly to WhatsApp.
* **Digital Invoices:** Customers can view and print their order receipts.
* **Order History:** Session-based history to track past delicious meals.
* **Admin Dashboard:** Secure panel to manage menu items and view pickup orders.
* **Interactive Elements:** Floating review bubbles and a "Today's Special" slide-in card.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Bootstrap 5), JavaScript, AOS (Animate on Scroll).
* **Backend:** Python (Flask).
* **Database:** SQLite (SQLAlchemy ORM).
* **Hosting:** PythonAnywhere.

## 🚀 How to Run Locally
1. Clone the repository: `git clone https://github.com/Rushi-1881/Mouli_bhojnalaya_Website_python.git`
2. Create virtual environment: `python -m venv venv`
3. Activate venv: `source venv/bin/activate` (Linux/Mac) or `venv\Scripts\activate` (Windows)
4. Install dependencies: `pip install -r requirements.txt`
5. Run the setup: `python setup_db.py`
6. Start the server: `python app.py`
